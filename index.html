GoPro Organizer

A shell script that organizes GoPro videos and photos

The script

The script, if you change the parameters, will move the media stored in a gopro camera (via USB or the SD plugged in the computer) to the desktop to a folder called goprohero (you can change that), it will create two folders inside the main directory, in this case goprohero. The folders are, obviously photos and videos. Inside photos it will create a directory called single for single taken pictures with the photo mode and also it will create a directory called timelapse-burst-continuous for pictures in burst mode, timelapse, and in continuous shot and in video+photo mode. For the folder video it will create four folders: single, for single videos. chaptered, for videos that are split because the recording is lareg or because looping video mode. thumbnails, for the video thumbnail files. And lowresvideos, for the low resolution video files

photos
  • single
  • timelapse-burst-continuous
videos
  • single
  • chaptered
  • thumbnails
  • lowresvideos

Replace /path/to/DCIM/XXXGOPRO/ by the path to the GOPRO directory, if you connect your GoPro via USB (only HERO3 and HERO2 cameras, HERO3+ Not supported over USB!) or connect the SD card to your computer (supported by HERO2, HERO3 and HERO3+), the path to GOPRO dir should look something like this: /media/konraditurbe/8765-4321/DCIM/100GOPRO/ An easy way to get the path is to drop a gopro photo to firefox and copy the path without the filename and the file:// protocol. Example:

file:///media/konraditurbe/8765-4321/DCIM/100GOPRO/GOPR0017.JPG

/media/konraditurbe/8765-4321/DCIM/100GOPRO/
If you have more than one XXXGOPRO folder, repeat ONLY lines 16 to 19 changing the XXXGOPRO folder number. This will move the contents to the project folder.

By default, the directory where all the videos and photos will be moved is called goprohero and will be located in the desktop. Change it if you want.

The code has been tested on Ubuntu linux terminal, but it works for Mac and other linux distros.
